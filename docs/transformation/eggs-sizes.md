## Recipe Conversion Note

A recipe stating **“UK Large Egg”** corresponds to:

- Europe: Large
- Canada: Extra Large
- United States: Extra Large or Very Large
- Australia: Jumbo
- New Zealand: Large to smaller Jumbo
- Post-Soviet: Average Select 0 (large size)
